# My ToDo List

A simple React application that displays a list of to-do items. 

## Features
  1. Displays a hard-coded list of todo items
  2. Demonstrates use of JSX, components, arrays, and .map()
  3. Can be extended by additional features, like adding or deleting todos

## Installation

 1. Clone the repo
    ```bash
    git clone https://github.com/IrynaPiatnochka/Todo-list
    
 2. Navigate the project directory
    ```bash
    cd my-todos-app
    
 3. Install dependencies
    ```bash
    npm install

## Running the development server

1. Start the development server
   ```bash
   npm run dev

   ```arduino
   http://localhost:3000
